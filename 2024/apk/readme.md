# APK

To install, drag file to emulator home screen. To uninstall, long press on the
app, and drag to "Uninstall". To force stop, long press on the app, and drag to
"App info". APK to Java:

~~~
jadx.bat --show-bad-code hello.apk
jadx-gui.bat com.pinterest-10098030.apk
~~~
